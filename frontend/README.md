# Crypto Trading Dashboard - Frontend

Modern React/Next.js dashboard for the adaptive grid trading bot.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Access at `http://localhost:3000`

## Build

```bash
npm run build
npm start
```

## Environment

Create `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Features

- ✅ Strategy management
- ✅ Real-time order tracking
- ✅ Trade history
- ✅ Portfolio overview
- ✅ Analytics charts
- ✅ Strategy controls
