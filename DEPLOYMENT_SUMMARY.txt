Start all services:
  docker-compose up -d

Stop all services:
  docker-compose down

Check status:
  ./scripts/docker-manage.sh health

View logs:
  docker-compose logs -f backend

Database access:
  ./scripts/docker-manage.sh sql

Get help:
  ./scripts/docker-manage.sh help
